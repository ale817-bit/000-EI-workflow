# visual-strategy

## 使用场景
- 已有叙事结构，需要形成统一视觉表达策略。

## 输入材料
- 叙事结构文档。
- `input/brand-assets/` 品牌规范。
- `input/references/` 视觉参考案例。

## 工作步骤
1. 提炼视觉关键词与风格方向。
2. 建立色彩、字体、图形、版式策略。
3. 定义导视/墙面/图表等应用原则。
4. 输出可评审的策略结论与评估标准。

## 输出格式
- 参考 `templates/visual-strategy-template.md`。
- 采用“策略说明 + 应用原则 + 评估标准”结构。

## 检查标准
- 品牌一致性明确。
- 风格方向可执行且可验证。
- 对 graphic system 阶段输入充分。

## 禁止事项
- 不得忽略品牌禁用规则。
- 不得仅给情绪化描述而无执行规则。
