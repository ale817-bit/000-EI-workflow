# archive-producer

## 使用场景
- 项目阶段结束，需要打包与移交成果。

## 输入材料
- `output/01` 至 `output/06` 的阶段输出。
- 评审结论与版本变更记录。

## 工作步骤
1. 盘点阶段成果与缺失项。
2. 编制交付清单并标注状态。
3. 编写归档说明与版本记录。
4. 检查文件命名、格式与可读性。

## 输出格式
- 参考 `templates/archive-readme-template.md`。
- 归档包内需包含 README 与交付清单。

## 检查标准
- 清单完整且可核对。
- 版本信息清晰。
- 外部成员可按说明快速定位文件。

## 禁止事项
- 不得归档未评审版本为最终版。
- 不得遗漏关键依赖或已知问题说明。
