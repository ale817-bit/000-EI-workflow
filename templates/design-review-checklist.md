# 设计评审清单（Design Review Checklist）

## A. 需求一致性
- [ ] 是否覆盖核心需求目标
- [ ] 是否回应关键业务诉求
- [ ] 是否标注待确认项

## B. 叙事质量
- [ ] 叙事主线清晰
- [ ] 节奏层次完整
- [ ] 信息层级明确

## C. 视觉策略
- [ ] 与品牌资产一致
- [ ] 视觉关键词可感知
- [ ] 风格不冲突、不失焦

## D. 系统可落地性
- [ ] 规范可执行
- [ ] 模块可复用
- [ ] 输出格式满足制作方要求

## E. 汇报有效性
- [ ] Storyboard 逻辑闭环
- [ ] 页面信息密度合理
- [ ] 讲述节奏可控
